Описание системы TM

Система предназначена для удобной работы и постановки задач удаленным сотрудникам. (дропам)

Основные возможности системы:

  • Регистрация в системе
  • Заполнение профиля сотрудника
  • Регистрация идентифицирующих документов в системе
  • Обмен файлами
  • Обмен сообщениями
  • Постановка задач сотрудникам
  • Комментарии к задачам

Список разделов системы

  • Registration
  • Dashboard
  • Staff
    • Todo List

Registration

Процесс регистрации в системе. возможности.

Система обладает гибкой регистрационной формой для новых сотрудников.

Существует возможность проводить регистрацию

  • С подтверждением телефонного номера / емейла сотрудника (SMS)
  • Без подтверждения

При регистрации задействованы следующие модули

  • Модуль автокоррекции телефонного номера и приведения его к международному формату
  • Модуль автокоррекции имени и фамилии пользователя и приведения их варианту с заглавными буквам
  • Модуль коррекции адреса / пре-популяции списков со штатами / городами.
  • Модуль выбора таймзоны для пользователя

Раздел Dashboard

Первичная страница для пользователя при авторизации в системе

Отображение в виде карточек

  • Мини профиль пользователя с фото и названиями департамента и должности, даты регистрации и текущего баланса
  • Мини профиль супервайзера для сотрудника в системе со ссылкой на профиль
  • Мини профиль компании, с указанием адреса, телефонов, емейлов. (в прогрессе)
  • Карточка со списком 5 последних / новых задач
  • Карточка со списком 5 последних / новых сообщений

Раздел Tasks

Возможности:

  • Постановка одиночной задачи сотруднику
  • Постановка массовой задачи нескольким сотрудникам
  • Изменение статуса одиночной задачи
  • Массовое изменение статуса задач
  • Оплата выполненной сотрудником задачи
  • Выборка задач конкретного сотруднику
  • Создание и редактирование темплейтов задач
  • Автоматическая отправка выбранных темплейтов задач новоприбывшим сотрудникам
  • Создание и редактирование лейблов для задач
  • Установка лейбла конкретной задаче
  • Массовая установка лейблов задачам
  • Добавление файла (-ов) к задаче

Экран "Список задач"

  • Просмотр общего списка задач и управление

Экран "Просмотр отдельной задачи"

  • Комментирование задачи как со стороны сотрудника, так и со стороны менеджера
  • Просмотр задачи, управление задачей, просмотр блока с технической информацией о задаче

Возможности:

  • Просмотр списка пользователей и менеджеров в системе
    • Индикация непрочитанных файлов, непросмотренных документов, невыполненных задач

    • Индикация статуса пользователя
  • Фильтрация по глобальным категориям пользователей
  • Фильтрация по меткам
  • Удаление пользователей
  • Массовый / единичный разрыв сессии пользователя

Массовые и единичные действия с пользователям

  • Добавление и удаление заметок
  • Добавление и удаление меток
  • Отправка задач
  • Отправка сообщений
  • Отправка SMS
  • Постановка задач саппортам / операторам на телефоне
  • Модификация баланса
  • Установка названия должности и департамента
  • Массовое или единичный разрыв сессии пользователя

Подраздел SMS LOG

Позволяет просматривать полный лог отправленных системой SMS сообщений.

Подраздел TODO LIST

Позволяет просматривать задачи, поставленные менеджером самому себе или другим менеджерам, касательно других пользователей системы.

Подраздел SEND SMS

Позволяет осуществлять массовую рассылку SMS по листу либо отправлять единичные сообщения на номера, которые не зарегистрированы в системе

System

Настройки системы

  • Изменения названия проекта (компании)
  • Таймзона, в которой будет работать система
  • Формат даты
  • Формат времени
  • Регистрация (вкл/выкл)
  • Контроль за дубликатами номеров телефонов (вкл/выкл)
  • СМС активация нового пользователя (вкл/выкл)
  • СМС уведомления новым пользователям (вкл/выкл)
  • Емейл уведомления новым пользователям (вкл/выкл)
  • Уведомления по протоколу XMPP для админа / менеджеров / операторов (вкл/выкл)
  • Режим обновления системы (вкл/выкл)
    • Настройка продолжительности режима обновления системы (в минутах)

В этом режиме будет отсутствовать окно авторизации в течение заданного времени
